## 网站性能优化项目

####Part 1: 优化 index.html 的 PageSpeed Insights 得分

1.打开tomcat
2.访问index.html
3.打开开发者工具检查，查看Pagespeed网页得分
4.根据PageSpeed提示进行了压缩图片，下载了已经压缩好的图片替换掉原图片，修改了图片的尺寸。
5.清除浏览器缓存后，重新进行pagespeed分析，得分为95分以上。
